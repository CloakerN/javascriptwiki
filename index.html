Variáveis em JavaScript

Em JavaScript, você pode declarar variáveis de três maneiras diferentes:

  • var: uma forma mais antiga de declarar variáveis, que tem escopo global ou de função.
  • let: permite declarar variáveis com escopo de bloco, sendo mais moderna e segura.
  • const: usada para declarar constantes, que não podem ser reatribuídas.

Arrays em JavaScript

            
Arrays são listas de valores. Aqui está como você pode criar e manipular um array:

            

const frutas = ['maçã', 'banana', 'laranja'];
console.log(frutas[1]); // Exibe 'banana'
            

        


        

            
Objetos em JavaScript

            
Objetos são coleções de pares chave-valor. Veja como criar um objeto:

            

const pessoa = {
    nome: 'João',
    idade: 30,
    profissao: 'Desenvolvedor'
};
console.log(pessoa.nome); // Exibe 'João'
            

        


        

            
Promises em JavaScript

            
Promises são usadas para lidar com operações assíncronas. Veja um exemplo:

            

const minhaPromise = new Promise((resolve, reject) => {
    setTimeout(() => {
        resolve('Operação concluída!');
    }, 2000);
});

minhaPromise.then(res => console.log(res)); // Exibe 'Operação concluída!' após 2 segundos
